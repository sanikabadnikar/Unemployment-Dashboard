📊 Unemployment Analysis Dashboard – Power BI
🔹 Project Overview
Developed an interactive Power BI dashboard to analyze unemployment trends across regions and time periods. The project focuses on deriving actionable insights to support data-driven decision-making in economic analysis.

🔹 Key Skills & Tools
Power BI • DAX • Power Query • Data Visualization • Data Cleaning • Data Analysis • Dashboard Design • Excel/CSV

🔹 Responsibilities
Collected and processed unemployment dataset from public sources
Performed data cleaning and transformation using Power Query
Created calculated measures and KPIs using DAX
Designed interactive dashboards with filters and slicers
Analyzed regional and temporal unemployment trends

🔹 Key Features
Region-wise and time-based analysis
Interactive slicers for dynamic filtering
KPI cards for key metrics (avg unemployment rate, trends)
Comparative visualizations for better insights

🔹 Business Impact
Enabled quick identification of high unemployment regions
Improved understanding of employment trends over time
Supported data-driven insights for policy and planning

🔹 Outcome
Delivered a user-friendly and interactive dashboard that simplifies complex unemployment data into meaningful insights for analysis and decision-making.
